# Tab3 生成参数位置和参考图锚定区布局重新设计

**问题描述**：Tab3 界面存在两个布局问题：
1. 生成参数（如模型选择、步数等）当前放在底部单独占一行，应挪到顶部工具栏右侧
2. 参考图锚定区需重新设计：
   - 左侧：显示文字锚定提示词（两行布局，上行灰色只读显示原始版本，下行为可编辑版本）+ 已选图片缩略图
   - 右侧：显示当集资产图片网格（当前集所有资产，不是仅当前组资产），点选后高亮显示

**触发场景**：用户在 Tab3 界面开发/迭代阶段，审查当前 UI 布局后提出的修改意见

**教训/建议**：
- 生成参数等高频控件应放在视觉焦点区域（顶部工具栏），而非底部
- 参考图选择区需要同时展示文字锚定和图片锚定，且图片范围应为全集资产而非仅当前组
- 锚定提示词需区分原始版和编辑版，用视觉差异（灰色只读 vs 可编辑）区分
